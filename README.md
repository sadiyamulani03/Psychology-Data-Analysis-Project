# 📊 Psychology Data Analysis Project (SPPU)

This project analyzes student performance data using Python and visualizes insights through multiple charts.

## 🚀 Features
- Data analysis using **pandas**
- Data visualization using **matplotlib & seaborn**
- No external dataset required (data stored inside code)
- Generates multiple charts automatically

## 📈 Visualizations Included
- Bar Chart (Pass Percentage)
- Line Chart (Student Trend)
- Histogram (Distribution)
- Pie Chart (Pass vs Fail)
- Heatmap (Subject-wise performance)
- Boxplot (Data distribution)

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

## ▶️ How to Run
```bash
  pip install pandas matplotlib seaborn
  python psychology_project.py
```
